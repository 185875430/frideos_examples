# various

Example app to show how to use some features of the [frideos](https://pub.dartlang.org/packages/frideos) package: 


* #### TimerObject: a simple stopwatch 
* #### StagedWidget
* #### AnimatedObject
* #### LinearTransition
* #### CurvedTransition
* #### Sliders
